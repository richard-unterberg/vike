Bun is a modern JavaScript runtime like Node or Deno. It was built from scratch to focus on performance and reliability. It uses the [JavaScriptCore](https://developer.apple.com/documentation/javascriptcore) engine (the engine behind Safari) and is built in Zig. Bun has broad npm compatibility.
